Think of this as a Linux Mint mirror with Windows to Linux migration-installation. Notice how this modified [installer](https://github.com/linuxmint/ubiquity) is completely separate from [Linux Mint](https://github.com/linuxmint/cinnamon) itself: which is why an installation of Linux Mint by this [migration tool](https://github.com/jackjumpinc/migratepc) is no different than standard installation. It just also copies over your user files and configures them as Linux Mint would if you had originated them there. The post-install script, if run, just installs and configures things you could do yourself on Linux Mint (with files copied over from Windows), like browsers you had installed on Windows. The post-install script also offers to install some other things not included in Linux Mint install that may or may not have been on Windows (optional: Steam, duplicity backups, Zoom, [Brave browser](https://github.com/brave/brave-browser)). Running post-install script or accepting a proposed option presented by post-install script is you choosing to add to or further embrace Linux Mint options (Steam is advertised during install but isn’t installed, Timeshift is installed but isn’t enabled).

The automatic partition screen of installer has been taken over by the migration tool – that’s where it’s found. Choose any other option and it’ll be just like the default installer as no changes have been made elsewhere – nothing will be migrated. Choose the automatic partition/erase entire disk, option and you won’t be able to proceed without selecting a copy drive in addition to an install drive for migration. One of the drives has to have Windows installed on it. The other has to be able to hold your user files alone or those files on Linux Mint. Do not proceed to install with this migration tool without carefully selecting both drives if you want to migrate something as you install Linux Mint. When you do proceed, the automatic partition action itself is no different. The gui interface was the only overlap and that small section of code is completely separate from actual automatic partition code which isn’t touched by migration code. Automatic partitioning just can’t be called anymore without migration parameters.

About seventy language options are supported by this migration-installation tool (less than installer). However, the post-install script added to main user home directory is only in English (even though it supports setting up non-English locales for those who selected a main system locale other than English in installation). The migration tool portion and post-install script will still work for those who choose a Linux Mint supported language that is not supported by Jackump.com, but migration related prompts and messages will be in English.
